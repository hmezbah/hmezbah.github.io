# Habibullah Mezbah — Premium Web Developer Portfolio

A modern, fast, and premium responsive portfolio website for **Habibullah Mezbah**, a professional Web Developer based in Dhaka, Bangladesh. Specializing in Laravel, WordPress, PHP, MySQL, React, JavaScript, and modern web solutions.

🚀 **Live Preview (Local Server)**: `http://127.0.0.1:8080/index.html`  
🌐 **Hosting Compatibility**: Fully optimized for static hosts like **GitHub Pages (`github.io`)**, Netlify, and Vercel.

---

## 🎨 Design Inspiration
Inspired by the clean grids, glassmorphism typography, smooth micro-animations, and theme toggling seen on premium tech sites like **Stripe, Vercel, GitHub, and Linear**.

## ✨ Features Built

- 🌓 **Dark & Light Mode Toggle**: Smooth theme transition utilizing LocalStorage memory and system preference overrides.
- ⚙️ **Static Form Emailer**: Handles validation in JS and delivers contact submissions directly to `hmezbah@gmail.com` using the **FormSubmit.co AJAX API** (no redirects, no captchas).
- 📱 **Fully Responsive Layout**: Fits smoothly on all screens (Desktop 3-column grid, Tablet 2-column, Mobile 1-column).
- 🏷️ **Dynamic Tag Filters**: Sorts projects on-the-fly (All, ERP, Laravel, WordPress, Corporate Website) with smooth transition animations.
- ⚡ **GPU-Accelerated Fades**: Subtle entrance animations that trigger when elements scroll into view.
- 💬 **Testimonial Slider**: Auto-rotating slideshow with navigation dot indicators.
- 🧮 **Counting Stats**: Years of experience and project completion metrics increment from 0 when scrolled past.
- 💬 **Direct WhatsApp Integration**: Fast-action WA links and vectors (`+8801722734209`) embedded in the header, footer, and contact section.
- 📄 **Direct Resume Tab**: Opens `Habibullah-Mezbah-CV.pdf` in a new tab immediately.
